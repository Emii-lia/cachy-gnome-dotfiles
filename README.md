# CachyOS Gnome dotfiles

## Prerequisites

Run the following command to install the required packages:

```bash
sudo pacman -S conky conky-manager2 sddm kando kitty neofetch paplay cowsay oneko xpenguins gnome-shell-extensions 
```

## Configuration

### Conky

After copying the conky configuration files to `~/.config/conky`, you can start Conky by opening the Conky Manager then selecting the Conky instances.
